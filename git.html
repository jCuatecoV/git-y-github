Iniciando en el mundo Git

Registro en la plataforma

  • $ git config --global user.name "Tu nombre de Usuario"
  • $ git config --global user.email "tu@email.com"

Comandos

Puntos importantes en el curso de GitHub

  • git log --online --> Visualiza todos los commits
  • git log -p --> Viasualiza a detalle las modificaciones realizadas
  • git log --author="Usuario"--> Visualiza solo las modificaciones del Usuario en especifico
  • git log --since=1.month.ago --until=1.day.ago --> Visualiza modificaciones de acuerdoa fechas
  • git log --pretty="format:%h %s" --> Visualiza mediante formato
